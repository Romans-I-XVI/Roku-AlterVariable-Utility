Installation

  • Add alterVariable.brs to your source folder.
  • Add these 3 lines to the start of your Main() function.
    • m.network_input = CreateObject("roInput")
    • m.network_input_port = CreateObject("roMessagePort")
    • m.network_input.SetMessagePort(m.network_input_port)
  • Call alterVariable() from an appropriate while loop.

Instructions

This script is for use by Roku developers. With this script you can execute code on an active Roku development channel.

  • First enter your Roku's IP address in the first field.
  • In the additional fields enter the code you would like to execute, each field is a line of code.
  • Press submit. That's it.
I hope you find this useful. I know when I'm making games I often want to make small changes and see how it looks/feels. I developed this so I could do that real time. :)
