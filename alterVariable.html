Instructions

This script is for use by Roku developers. With this script you can change the value of any variable that has been declared with the special "m" pointer. This could be either a global variable if called outside an associative array or any variable inside the associative array that has been used to call the function. "m" The Brightscript "this pointer"

  • First enter your Roku's IP address in the first field.
  • In the second field set the name of the variable you want to change. For example if your Brightscript says --- m.aa = {one: 1, two: 2, three: 3} --- the variable field on this page could say --- aa.three ---. Similarly you can change a value in an array by using brackets. For example if your Brightscript is --- m.array = [0,1,2] --- to change the 0 index the variable field on this page should say --- array[0] ---.
  • In the third field you can set what you want the variable to change to. Currently this can be a string, integer, float, or array.
I hope you find this useful. I know when I'm making games I often want to make small changes and see how it looks/feels. I developed this so I could do that real time. :)
