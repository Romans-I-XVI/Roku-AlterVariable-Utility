Instructions

This script is for use by Roku developers. With this script you can change the value of any variable that has been declared with the special "m" pointer. This could be either a global variable if called outside an associative array or any variable inside the associative array that has been used to call the function. "m" The Brightscript "this pointer". To install just add the file alterVariable.brs to your source folder and call alterVariable() from an appropriate while loop.

  • First enter your Roku's IP address in the first field.
  • In the second field set the name of the variable you want to change. Note that the brightscript uses "." as a dilimiter. For example if your Brightscript says --- m.aa = {one: 1, two: 2, three: 3} --- the variable field on this page could say --- aa.three ---. Similarly you can change a value in an array by using brackets, however opposite normal convention you must leave the "." as it is the dilimiter. For example if your Brightscript is --- m.array = [0,1,2] --- to change the 0 index the variable field on this page should say --- array.[0] ---.
  • In the third field you can set what you want the variable to change to.
  • Lastly make sure the check whether the value is a string or an integer (also check integer if it's a float).
I hope you find this useful. I know when I'm making games I often want to make small changes and see how it looks/feels. I developed this so I could do that real time. :)

*Note - the variable is currently limited to 10 levels of array/associative array "." delimiters.
